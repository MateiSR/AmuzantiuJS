# Amuzantiu

Amuzantiu is a Discord bot written in Discord.js for small communities.

## Installation
Node.js version ``>=16.9``

```bash
npm start
```

## Usage
Set TOKEN environmental variable to your bot's:
```powershell
$Env:TOKEN = ""
```
